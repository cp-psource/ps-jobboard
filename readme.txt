== Description ==

Veröffentliche einen Job
Veröffentliche Projekte über eine intuitive, reaktionsschnelle Front-End-Oberfläche. Beschreibe Dein Projekt, lege ein Budget fest, setze eine Frist und hänge Spezifikationsdateien an. 
Erstelle eine Webseite, auf der Qualitätsprofis mit Dir an Deinem nächsten großen Projekt arbeiten möchten.

Präsentiere Deine Fähigkeiten
Es hat alles, was Du brauchst, um Deine Fähigkeiten zu zeigen. Teile einen Avatar, eine Biografie, Kontaktinformationen, Beispielarbeit und einen Link zu sozialen Profilen. 
Wähle die perfekte Person von Hand aus, basierend auf Erfahrung, Portfolio und Empfehlungen anderer.

Aktiviere Erweiterungen 
Und erweitere die Möglichkeiten von PS Jobboard um weitere Funktionen, so kannst Du zum Beispiel über die MarketPress Schnittstelle Guthabenpakete anbieten um Dein Jobboard zu monetarisieren.
